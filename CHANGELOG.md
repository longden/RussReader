# Changelog

All notable changes to RSS Reader will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-01

### Added
- Initial release
- Menu bar RSS/Atom feed reader
- Multiple feed subscription support
- Smart filtering (All, Unread, Starred)
- OPML import/export
- Customizable appearance (font size, colors, refresh interval)
- Keyboard shortcuts (⌘R refresh, ⌘, preferences, ⌘Q quit)
- Default feeds (Daring Fireball, Swift by Sundell, NSHipster)
- Persistent storage via UserDefaults
- Localization support (English)
